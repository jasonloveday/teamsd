Term Definition: activity
Main Description

In the UMA , an activity is a breakdown element which supports the nesting and logical grouping of related process elements such as descriptor and sub-activities, thus forming breakdown structures.

Generally, an element of work to be performed during the course of a project.

More specifically, in the context of a work breakdown structure (WBS) organized in a structured way, an activity is an element of work at a particular hierarchical level.
