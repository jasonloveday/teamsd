Guideline: Choosing and Using an Estimating Approach
The purpose of this document is to help you choose and use appropriate estimating approaches when carrying out an estimate.
Relationships
Related Elements
  • Estimation Report
Main Description

There are 7 approaches outlined in this document 2 of which describe typical specialist estimating tools.

The approaches are aimed at supporting the creation of estimates in software development; however, most of the listed approaches may be tailored to work with virtually any type of project.

The paper should be read by anyone who is required to create an estimate, e.g. Project Managers, IT Architects, Business Analysts and others. Note that the words Estimating and Estimation are used interchangeably, to mean the same thing.
