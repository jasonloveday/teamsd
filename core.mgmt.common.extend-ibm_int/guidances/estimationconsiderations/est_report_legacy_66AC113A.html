Estimation Considerations: Project Estimates

Main Description

Initially 2 to 8 hours will be needed to develop the measurements for key metrics and to perform a top-down estimate. Based on this, determine how much further effort should be expended to refine the estimate by going on to a bottom-up approach, balancing the accuracy of available information and the effort in obtaining it against the accuracy that is needed for the estimate.

In the Solution Outline or Engagement Startup phase:

Use the proposal estimate as a benchmark. Calculate the effort to produce this work product using the following formula. Allow about 0.5% of the previously estimated total project effort for an estimate covering the entire project, or up to 2 % of the Solution Outline phase effort for a phase only estimate.

In the Macro or Prepare phase:

Allow about 0.5% of the total project effort (as estimated before), or up to 2 % of this phase effort for a phase only estimate.

In Micro Design or Redesign/Design phase:

Allow about 0.5 % of the total project effort (as estimated before) and about the same amount for each release.
