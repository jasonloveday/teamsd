IT Standards for XYZ Company

Introduction
The purpose of this document is to describe the preferred/allowed technical components for any new solutions developed for the XYZ Company. The document is organized by class of product or component.

Database
In May 1995, the CIO commissioned a study of available databases for the purpose of pre-selecting a product to be used for all new development. The recommendation of DataMaster product was made by the project team and was accepted in July of 1995.
Since this time, all new application development has been done using this product. While there are still legacy data stores in use, these are being replaced by attrition. Currently, 75% of the data of XYZ Company is stored in DataMaster. The IT Operations staff is fully trained to operate the product. The applications development staff is fully trained to write and maintain DataMaster applications.
As of November 1997, XYZ Company has installed DataMaster Version 1.71, release G, for all applications in production.

Server Hardware
XYZ Company does not have a preferred server hardware environment. The current installed base consists of: 

  • IBM RS6000
  • Hewlett Packard
  • Digital Alpha

For new applications, server hardware will be determined by application performance requirements, among these three vendors. XYZ intends to focus on two of these three vendors over time, to control administrative and operating costs of the server hardware.
