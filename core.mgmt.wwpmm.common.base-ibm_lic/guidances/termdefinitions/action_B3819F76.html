Term Definition: action
Main Description

A unit of work that must be performed by the project that was neither included in the project work plans nor of sufficient impact to warrant a change to the project schedule.

Actions can be initiated for many reasons, such as a corrective action in response to a compliance issue or a risk planning action in response to a risk being identified.
