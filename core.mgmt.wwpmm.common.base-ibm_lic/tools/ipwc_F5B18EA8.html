Tool: IBM Program Work Center (IPWC)
This tool provides comprehensive project and program management capabilities required for managing services engagements.
Main Description

IBM Program Work Center (IPWC) is recommended for project and program management including all services organizations, or organizations offering services as a part of their business model. This tool provides comprehensive project and program management capabilities required for managing services engagements.

For the latest information on IPWC deployment, education and other references, please go to the Project Services website/IPWC.

If you want to view and share experience, please join the IPWC Community of Practice, which also includes reference materials.

To view training and templates created and in use by GBS, take a look at the GBS PM&T IPWC "Jump Start" Sessions.
