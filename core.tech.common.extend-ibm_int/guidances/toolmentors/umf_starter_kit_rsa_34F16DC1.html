Tool: Rational Software Architect
Main Description

The following link points to an asset in iRAM which references a ZIP-file containing a directory structure and a RSA (Rational Software Architect) template based on a set of IBM Unified Method Framework architecture artifacts. The directory structure is a starting point for project teams to organize their files according to the standard domains and work products of UMF.

The RSA template (called 0RootModel.emx) contains a UMF-like tree structure and provides copy/paste templates for the following Artifacts:

  • System Context (ART 0651)
  • Use Case Model (ART 0508)
  • Component Model (ART 0515)
  • Deployment Units (ART 0525)
  • Operational Model (ART 0522)

It was created with SDS (IBM System Description Standard, previously called ADS, Architecture Descriptions Standard) in mind, i.e. because of RSA limitations not all diagrams are truely SDS-compliant, but this is a proof-of-concept that RSA can already be used to design the functional and operational architecture of an IT System.
