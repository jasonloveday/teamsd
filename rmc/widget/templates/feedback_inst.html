Browser instructions

Netscape Communicator 4.x:

Netscape Messenger is the required email component of the Netscape Communicator package. If you have Netscape Navigator only, you will need to install Netscape Messenger.

  1. In the top menu bar of Netscape, click Edit, then Preferences
  2. Expand Mail & Newsgroups
  3. Select Identity
  4. Using your Lotus Notes shortname, fill in your email address (e.g., jdoe@us.ibm.com)
  5. On the left side, click Mail Server
    For Outgoing mail (SMTP) server enter a mail server as follows:

    If you are located in Europe, and served by the United Kingdom data center, your SMTP server name is relay.uk.ibm.com

    If you are located in Europe, and served by the German data center, your SMTP server name is relay.de.ibm.com

    If you are located anywhere else (e.g. US, Canada, AP, etc.), your SMTP server name is relay.us.ibm.com

  6. Click OK

 

Netscape 6.x:

Netscape Mail is the required email component of the Netscape 6.x package. If you have Netscape Navigator only, you will need to install Netscape Mail.

  1. In the top menu bar of Netscape, click Tasks, then Mail
  2. In the top menu bar click Edit, then Mail/News Account Settings
  3. Fill in the Identity section using your email address (e.g., jdoe@us.ibm.com)
  4. On the left side, click Outgoing Server (SMTP)
  5. For Outgoing mail (SMTP) Settings enter a mail server as indicated above (Netscape 4.x instructions)
  6. Click OK

MS Internet Explorer 4.x or higher:

  1. In the top menu bar of MSIE click Tools, then Internet Options
  2. Click the Programs tab
  3. Move to the E-mail window and click the down arrow to view options
  • If Lotus Notes is an option, choose it
  • Otherwise, select an e-mail program, then click Apply in the lower right corner of that dialog box
  1. Click OK
