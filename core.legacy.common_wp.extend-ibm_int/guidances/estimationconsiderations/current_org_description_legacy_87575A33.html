Main Description

Plan for 5-30 days depending on the size and complexity of the project. It is important that you leverage client resources to gather data. If using any of the following tools, plan additional time.   

If the Denison Survey is used, allow 3-4 consulting days for planning and preparation, analysis, and report writing. Add sufficient elapsed time to ensure a good response rate. The elapsed time should take into account the initial response timeframe, a follow-up to obtain more responses, forwarding to Aviat (the processors), and Aviatâ€™s processing turnaround (within one week).   

If the Managerial Style Questionnaire (MSQ) is used, build into the plan a half-day client workshop and 2-3 days of subsequent analysis.   

If the Organization Network Analysis (ONA) is used, build into the plan about 5 days of consulting time for planning, preparation, analysis, and report writing. Survey input is a time-consuming task--a few automated tools are being piloted which will soon be published in intellectual capital, but we recommend obtaining secretarial support for this activity. Feedback sessions, if used, will add one day each. See the ONA Sharing Room (link above) for a sample proposal and pricing guidelines.
