Tool: Rational PurifyPlus
This tool is an advanced code debugging and diagnostic tool.
Main Description

IBM® Rational® PurifyPlustm application is an advanced debugging and diagnostic tool that will do the following:

  • pinpointing hard-to-find, run-time errors in your application;
  • providing advanced application performance profiling;
  • pinpointing areas of code that have not been exercised during runtime execution

While Rational PurifyPlus is typically used by developers during their test and debug activities, it can also be incorporated into the testing activities performed by independent testers.
