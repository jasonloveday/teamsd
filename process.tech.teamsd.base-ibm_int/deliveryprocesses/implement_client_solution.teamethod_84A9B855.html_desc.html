Activity: IMPLEMENT Client Solution
This activity covers the development/integration of the solution and transition to delivery.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose
The purpose of this activity (CVM phase 4) is to ensure the successful implementation of the solution according to agreed-to criteria.  Starting with the client's value drivers and expectations for the opportunity, the team DELIVERS the value as described in the proposed solution and in the manner in which we help the client implement it.  The progress indicator for this activity is the completed implementation of the solution. 
Relationships
Parent Activities
  • Support Implementation and Confirm Value
Description

At this point in the buying process, the client implements the solution and monitors against project plan and original business case to validate success. Where appropriate, the ITA will work with the implementation team to ensure success, gain skills, and demonstrate commitment. This is also the time to harvest assets for reuse on later projects, at this client or to be shared with other IBM teams.

The following are done during this Activity with involvement by technical sales and, where appropriate, services:

  • Finalize client satisfaction/success criteria and plan.
  • Develop detailed project plans.  Manage the contract/ project.
  • Implement  solution capabilities to achieve desired performance, impact and benefit.    
  • Build the client â€˜s solution skills.
  • Initiate and Launch, Execute, Close.  Deliver the products and services per the contract.
  • Communicate progress and status.
  • Resolve  issues thru Delivery Single Point of Contact (SPOC).

When services are involved, we can verify our value when the client accepts our deliverables and confirms value received.  The payment is received, project resources are released and the contract is closed.

At the end of this Activity, the proposal is implemented and the client receives the expected value.
