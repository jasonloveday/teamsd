Guideline: Documenting Client Relationships
This guideline provides guidance on a possible source for information required in performing a task to describe the current organization.
Relationships
Related Elements
  • Describe Current Organization
Main Description

Finding necessary information about client relationships can take a significant amount of time. Assuming the IBM Account Plan is in place, it has a section called Client Relationship Responsibilities which is used to document who in IBM has responsibility for which client.  It can be used with Current Organization Description to get a more complete picture of the client. The following material shows the directions to the account team and illustrations from the Account Plan.

Excerpt from Account Plan:

Record this information for the current relationship in the Last Year (LY) column and your goals for the relationship in the This Year (TY) column.
