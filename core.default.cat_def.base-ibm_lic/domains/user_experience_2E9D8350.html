Domain: User Experience
The work products in the User Experience sub-domain address the design, development, and management of the user interface that defines the entire experience of the interaction between the end user and the system or environment.
Main Description
The work products focus on four key areas: user interface planning, design, implementation, and measurement. They are also use to evaluate the effectiveness of user interfaces so that they meet expectations.
Key Considerations
User interfaces include not only graphics, text, and navigation on a screen but also interaction with other devices and environments that affect user interaction such as RFID transactions and automated facilities.
