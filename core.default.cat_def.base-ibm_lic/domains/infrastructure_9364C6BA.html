Domain: Infrastructure
The work products in the Infrastructure domain define the necessary supporting framework required to deliver a solution in the form of facilities, networks, operating platforms, workstations, servers, and storage.
Main Description
The Infrastructure work products are the physical implementation of an architecture.  These work products include the specification and configuration of resources to provide a solution.
