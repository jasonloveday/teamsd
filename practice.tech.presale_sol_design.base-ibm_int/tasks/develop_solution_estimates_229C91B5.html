Task: Develop Solution Estimates
This task will develop solution estimates when necessary for inclusion in a proposal.
Disciplines: Schedule Management
Purpose
The purpose of this task is to develop estimates for hardware, software and technical and non-technical effort for the purpose of developing a solution proposal for the client.

      Main Description
      In this task, estimates are developed for hardware, software and technical and non-technical efforts. This is accomplished by first identifying the scope of estimation to determine what skills will be required. An estimation approach is selected and applied to the proposed infrastructure and any effort (services) that will be proposed. Where services are required, a schedule is also necessary to enable estimation of the technical and non-technical effort.
      Steps
      Evaluate Type of Estimates Required
      Determine whether the estimates are limited to hardware and software only, or whether estimates will also be required for technical and non-technical effort. If estimating involves technical and non-technical effort, consider adding an estimating subject matter expert to assist in the remaining steps.
      Select Estimating Approach
      Select estimating approach. This step assumes that the estimate is non-trivial and as such will require the assistance of an estimating subject matter expert.
      Estimate the Technical Effort
      Estimate any technical effort and all non-Technical effort (such as Project Management, travel and living) for the chosen estimating approaches, including effort associated with mitigating any risks identified.
      Calculate Project Schedule

      Calculate the schedule associated with this effort. Consider the use of scheduling tools such as Rational Portfolio Manager (RPM) or Microsoft Project

      • Use the effort estimates and add resource assignments to develop the schedule, documenting any estimating and resource utilization assumptions, and record the estimated schedule information.
      • Ensure that resource has been allocated to all tasks, and has been levelled.
      • Use current resource availability and known calendar events (such as holidays) to calculate the schedule for the total project effort.
      Calculate Resource Costs Based on Overall Schedule and Effort
      Calculate the resource costs based on overall schedule and effort. To calculate resource costs, multiply the rate at which each project team member will be charged by the number of hours that are estimated for that resource. Many organizations use a blended rate that applies to all resources, in which case the cost calculation is simply the total estimated person hours times the blended rate.
      Estimate Hardware and Software Costs
      Estimate the Hardware and Software costs and any additional aspects of costs that have not yet been considered, for example Licenses, Critical Computer resources, Intellectual Capital and so on. See estimating considerations checklist.
      Document the Estimates
      Document the estimates (size, effort, schedule and cost) in the estimation report using the guidance given in the template.
      .
      Key Considerations
      This task may not be required in simple situations, especially where no services will be proposed.
