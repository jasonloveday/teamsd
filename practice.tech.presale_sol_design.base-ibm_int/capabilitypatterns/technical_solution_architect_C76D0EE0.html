Role: Technical Solution Architect (041627)
This role assumes overall technical responsibility for the success of solution construction, implementation and system integration in a technology, industry or business speciality.
Relationships
Performs:
  • Define Key Services
  • Develop Architecture Overview
  • Develop High Level Component Model
  • Develop High Level Operational Model
  • Develop Solution Estimates
  • Evaluate Integrated Solution
  • Identify Delivery Approach
  • Propose Solution, Resolve Concerns
  • Refine Viability Assessment
Responsible for:
  • Architectural Decisions
  • Architecture Overview
  • Business Direction
  • Candidate Asset List
  • Component Model
  • Estimation Report
  • Non-Functional Requirements
  • Operational Model
  • Project Definition
  • Requirements Matrix
  • Service Model
  • Standards
  • System Context
  • Technical Environment
  • Use-Case Model
  • Viability Assessment
Additionally Performs:Modifies:
  • Architecture Overview
  • Component Model
  • Estimation Report
  • Operational Model
  • Project Definition
  • Service Model
  • Viability Assessment
Main Description

This role delivers new and complex high quality solutions to clients in response to varying business requirements. They utilize in-depth knowledge of leading-edge products and technology in conjunction with industry and business skills to influence clients.

Staffing
Skills
  • Apply Client Strategy and Planning
  • Apply Contract Bid and Negotiation Management
  • Develop Client Solutions
  • Develop Proposals
  • Manage Non-Standard Contract Terms
  • Use Financial Techniques
