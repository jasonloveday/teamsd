Task: Propose Solution, Resolve Concerns
This task will ensure both the customer and IBM reach final agreement on how the solution will be implemented in order to produce a final proposal and a signed agreement.
Purpose

Main Description

Prior to a formal proposal, there is usually a review done, at least by the Client Relationship Team, of the entire proposal.  This includes not only the technical aspects, that the Technical Solution Architect is responsible for, but also the non-technical aspects.  If the project is large and complex, then a Proposal Baseline Assessment will be completed.  In this case, prior Technical Delivery Assessments (or Solution Assurance Reviews) and the Integrated Technical Review will be input to the PBA.  See the Guideline below for further information on this.

Once the proposal is made, there are usually questions and concerns that must be overcome.  From a technical point of view, perhaps a proof-of-concept, test for indicators of performance, or other "proof" is required.  If so,  it is done during this task.

Client concerns might also cause a change in scope or the phased implementation.  Perhaps the client benefits aren't in line with the costs.  Maybe there are client inhibitors, like availability of skilled resources, that have to be accommodated. The changes will require a careful review to ensure that we still have a viable project with appropriate value to the client.

The Project Definition has useful information to help with this process since it captures the what, when, who, why, where and how of the project.  Obviously, any changes will need to be reflected in an updated Project Definition (and probably Viability Assessment). 

This task is completed when:

  • We have a signed contract, statement of work, and
  • Implementation team and dates have been established.
Key Considerations
The Technical Solution Architect will ensure that the proposal is technically viable.  There is a balance to be made here -- to provide just enough detail to assure the client that requirements are understood and solution will address them.
