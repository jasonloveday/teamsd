Capability Pattern: Pre-sale Solution Design
This capability pattern is intended for use as the pre-sale solution design phase, it may be incorporated in any pre-sale delivery process where high level solution design is required.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Purpose

This capability pattern is intended for use as the pre-sale solution design phase. It uses tasks and activities that are tailored to the pre-sale context and which correspond to the EXPLORE and DEVELOP phases of the Client Value Method (CVM). Work products used are standard UMF work products. It may also be incorporated in any pre-sale delivery process where high level solution design is required.
