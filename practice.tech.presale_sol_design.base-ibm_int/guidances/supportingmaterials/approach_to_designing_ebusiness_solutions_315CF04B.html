Main Description

This Redbook is not current and contains some obsolete information but is retained as reference because it contains certain helpful guidance on solution reviews, Proofs of Concept (POC) and Test for Indicators of Performance (TIP). TIP is a technique for limiting the performance testing effort to only what is needed to validate the solution. Information on the POC and performance testing is found in appendices A and B.
