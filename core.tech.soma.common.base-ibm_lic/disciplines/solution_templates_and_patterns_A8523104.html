Discipline: Solution Templates and Patterns
Today's client solutions are often a hybrid of multiple solution types. SOMA codifies these best practices of combining these solution types in a hybrid approach to developing solutions based on service-oriented principles in Solution Templates.
Main Description
Today's client solutions are often a hybrid of custom development, leveraging legacy applications and web services enabling them, integrating and using adaptors, including packaged application integration. Each of these have found to have recurrent themes in their applications to SOA. SOMA codifies these best-practices of combining these in a hybrid approach to developing solutions based on service-oriented principles in Solution Templates.
