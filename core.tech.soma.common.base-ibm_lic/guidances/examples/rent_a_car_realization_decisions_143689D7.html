Table 1 documents the realization decision that was made related to the support of an IMS fixed message format.

Issue

The IMS fixed message format to integrate with legacy IMS based COBOL and Assembler language application programs cannot be changed.

Guiding Principles

         
  • Maximize delivery of business capability within time/money constraints          
  • Minimize impact of change to Reservation System and existing POS          
  • Minimize technology churn, system integration and host development risk          
  • Leverage the existing fixed message format to access legacy applications

Options Considered

Option: MQ IMS Bridge

This option would allow services to be exposed using an XML message interface, but the messages will be transformed to IMS fixed format before presenting to legacy systems.
This will be achieved through MQSeries based message transport using MQ IMS bridge. The SOAP messages are converted by MQ IMS bridge to proprietary fixed IMF format for the backend systems. Similarly, it does reverse transformation from fixed IMF format to XML SOAP messages.

Realization Decision

Expose services using an XML message interface but transform message to IMS format before presenting to legacy system.

Justification

  • It preserves the IMS fixed message format for communications with legacy system while exposing the services via XML. No other option was considered due to the constraints:
    • IMS fixed message format must continue to be used for interactions with legacy system
    • MQ Series message transport must be part of any solution that requires services to be exposed

Implications

If transformation is to be done on zSeries, a new technical capability would need to be introduced into that environment. This would violate a basic principle of the project: “avoid impact to the existing applications and environment”. XML would need to be transformed to IMS fixed format messages at some point in the flow.

Transformation will be performed on a distributed platform before/after zSeries.
