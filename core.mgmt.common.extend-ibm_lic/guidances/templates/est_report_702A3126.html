Template: Estimation Report
This template contains a Project Details worksheet, a minimum of two approach worksheets plus a baseline worksheet for the Technical Estimate and for the Project Estimate, plus a Price worksheet. This template is intended for a software project, however this can be customized for other project types.

Main Description

The Estimation Report artifact contains different blocks of worksheets â€“ a project details worksheet and one block of worksheets for each governance point in the estimate lifecycle. An estimate has three governance points: the Independent Technical Review, the Independent Project Review, and the Price Review.

The Baseline worksheet corresponding to that governance point is completed to document the details of the estimate being passed forward for review. The worksheets are updated based on the feedback from the review and become the baseline estimate.

The appropriate worksheet (either Technical Estimate: Approach or Project Estimate: Approach) should be completed for each estimating approached used.  Best practices require a minimum of two different approaches; each approach is documented in its own worksheet.  The approach worksheets should be copied if more are needed.

As a result of reviewing and discussing the results of each approach used, a decision is made to determine the baseline for that estimate. This baseline is recorded on the corresponding Baseline worksheet.

Additional more detailed instructions on completing this template are included on the Project Details worksheet.
