Since links to available assets are fragile, we have moved the content of this guideline to the TeamSD web site under "Where to Get Help" http://w3-03.ibm.com/support/tsdesign/help.html

At this web page, you will find available assets and other information that will help you design a client solution with TeamSD.  At the time of the creation of this guideline, this page had information in the following categories:

  • Technical Sales Accelerators
  • iRAM
  • Solution Gateway 
  • e-business Patterns
  • Expertise Locator

The Assets module in the Arrchitecture Thinking class provides further practical guidance on the approach to Asset Reuse and the primary sources for IBM IT Architects.

Mobile Access for Technical Sales Accelerators

Solution Seeker is a new mobile and web app for the field to simplify finding solution and solution design content. It provides an intuitive and streamlined interface and delivers curated information from across IBM. The Solution Design Accelerators delivered through this app are actively used by both general and technical sellers and are aligned with the stages of our sales process.

Solution Seeker is available as a native iOS app via the MaaS360 App Store. It is also available as a mobile web app at https://mobile.us.ibm.com:15044/soe/apps/services/preview/SoEApp/common/0/default/index.html.
