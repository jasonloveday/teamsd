Tool Mentor: UML Tooling for Team Solution Design (StarUML)
This tool mentor describes the use of StarUML as a potential tool for use in UML diagramming within Team Solution Design.
Main Description

Although there may be times when our full function tooling (like Architect's Workbench or Rational Software Architect) might be needed, there are also times when we just need a simple tool to create work product diagrams.  StarUML is an open source tool that is light weight and easy to use for drawing UML diagrams. StarUML is an open source project to develop fast, flexible, extensible, featureful, and freely-available UML/MDA platform running on Win32 platform. The goal of the StarUML project is to build a software modeling tool and platform.  StarUML diagrams can also be saved in .emf format and opened by Visio. StarUML is not as full function as the Rational tools but can be very useful when your purpose is to get a bit of help in drawing Team Solution Design diagrams.
