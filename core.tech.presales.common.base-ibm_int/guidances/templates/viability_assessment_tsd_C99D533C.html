Main Description

The attached templates are intended to record the Risks, Assumptions, Issues and Dependencies (sometimes referred to as the RAID log).  For simple projects, the document template (Viability_Assessment_Template_for_Team_Solution_Design.doc) has been supplied to capture this RAID log.  For more complex projects, with many RAID log entries, the spreadsheet version (Viability_Assessment_Template_for_Team_Solution_Design.xls) will be more useful. 

Also note that it may be necessary to produce a separate RAID log to capture items that you don't wish to review with the client (like IBM internal issues being resolved).
